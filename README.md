# DIGIHAZ PhD Course — Shared Datasets

Datasets are organised by type. See each sub-folder README for file catalogue and licences.

| Folder | Contents |
|--------|----------|
| `seismic/` | Earthquake waveforms, catalogues, RINEX files |
| `hydro/` | River levels, rainfall, flood extent records |
| `geospatial/` | Shapefiles, DEMs, satellite imagery |
| `iot_sensor/` | IoT time-series data |
| `socioeconomic/` | Exposure and vulnerability data |
| `synthetic/` | Synthetic datasets for exercises |

**Large files (> 50 MB):** upload to the DIGIHAZ shared drive and add a `.url` pointer file here.
